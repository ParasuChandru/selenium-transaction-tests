# Selenium Automation Suite: Create Transaction

This repo contains Page Object Model-based Selenium tests for the "Create Transaction" feature.

## Structure

- pages/         # All page object classes
- tests/         # Suite of test case(s)
- requirements.txt

## Running the Suite

1. Install Python dependencies:
    pip install -r requirements.txt

2. Adjust the base_url, test credentials, and selectors as per your app.

3. Run all tests:
    python -m unittest tests/test_create_transaction.py
